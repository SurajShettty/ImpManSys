# UI / UX Specification

## Navigation

-   Dashboard
-   Clients
-   Projects
-   Templates
-   Calendar
-   Reports
-   Administration

## Dashboard

Cards: - Active Projects - Delayed - Go-Live This Month - Pending
Approvals - Risks - Issues

Charts: - Project Status - Resource Utilization - Module Progress -
Client Health

## Client Screen

Tabs: - Overview - Projects - Documents - Communication - Usage -
Settings

## Project Screen

Tabs: - Summary - Modules - Timeline - Tasks - Risks - Issues -
Meetings - Documents - Activity

## Module Screen

-   Phase accordion
-   Task list
-   Progress bar
-   Checklist

## Task Drawer

Fields: - Description - Assignee - Reviewer - Dependency - Due Date -
Checklist - Attachments - Comments - History

## Reports

-   Project Summary
-   Delay Report
-   Resource Report
-   Client Health
-   Adoption Report

## UI Guidelines

-   Color coded RAG status
-   Drag-and-drop Kanban
-   Gantt timeline
-   Global search
-   Advanced filters
-   Export to Excel/PDF
-   Dark mode support (optional)
